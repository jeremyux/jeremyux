CEO App

IRAM

This was a fun one-off project I did for a start-up company based out of India called IRAM. Their mission statement: to develop a tool for CEOs so they can get visual data and metrics of their hospital operations.

Dashboard View
Onload the app will direct the user to the Dashboard view. From here the user can navigate through the dashboard pagination via top nav-chips, while at the bottom the user can navigate at the parent level via a fixed bottom bar. Here the user will be able see a high-level view of cash to insurance ratio.

Dashboard View Scrolled
Additionally this dashboard will show an overview of: notifications, admissions of the day, bed vacancy, operating room operations, laboratory operations and radiology options.

Notifications
Entrypoints to the notification section can be found on the dashboard overview as well as the top nav-bar. From the notification screen the user will see important messages/ notifications from respective departments of the hospital.

Revenue
Lastly the user can see up-to-date reports of revenue from the following streams: submissions, receipts, resubmissions and denials.
